# 🍶 白酒行业财务指标分析系统

## 🎯 项目简介
一个完整的白酒行业财务分析系统，覆盖11家上市公司、291个财务指标，结合AI智能分析和交互式数据可视化。

## ✨ 核心功能

### 📊 全面数据覆盖
- **11家白酒上市公司**：泸州老窖、古井贡酒、酒鬼酒、山西汾酒、五粮液、洋河股份、今世缘、舍得酒业、水井坊、口子窖、贵州茅台
- **291个财务指标**：9个维度，全面分析企业财务状况
- **10年数据**：2014-2024年完整时间序列

### 🤖 AI智能分析
每个指标组包含AI深度分析：
- **直接分析**：基于阈值标准的时间趋势评估
- **间接分析**：结合白酒行业特点的深度解读
- **分析结论**：简洁明了的趋势判断和建议

### 📈 交互式可视化
- **多种图表类型**：折线图、柱状图、象限点阵图
- **时间序列探索**：查看任意年份数据
- **公司对比**：快速切换不同公司

### 🔍 智能搜索
- 按公司名称搜索
- 按指标代码搜索
- 快速定位目标数据

## 🎮 使用指南

### 基本操作
1. **选择公司**：从左侧目录选择公司
2. **选择指标**：从目录选择指标组
3. **查看分析**：右侧显示图表和AI分析
4. **切换年份**：查看不同年份数据

### 演示示例
**分析泸州老窖的核心利润**：
1. 选择公司：泸州老窖 (000568.SZ)
2. 选择指标：7.11 核心利润与核心利润率
3. 查看结果：
   - 📈 2014-2024年利润趋势图
   - 🤖 AI分析卡片（直接/间接/结论）
   - 📊 详细数据表格
